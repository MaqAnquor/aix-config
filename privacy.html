Privacy Policy

Last updated: 28 June 2026

Your conversations never leave your browser. We don't collect, store, transmit, sell, or see any of your data. There is no account, no server, no analytics, no tracking.

What the extension does

AI Chat Exporter reads the AI-chat conversation on the page you're viewing and converts it to a file (Markdown, etc.) entirely on your own device. Reading the conversation, formatting it, copying or downloading it — all of it happens locally in your browser.

What we collect

Nothing. No personal data, no conversation content, no usage analytics, no identifiers. We have no backend to send anything to.

Network activity

The extension makes exactly one network request, and it sends none of your data: it fetches a small public configuration file (CSS selector definitions — data, not code) from GitHub so the exporter can self-correct when a site changes its layout. This is a plain download containing no information about you or your conversations. If it fails, the extension simply uses its built-in defaults.

Permissions, and why each is needed

  • activeTab — to read the conversation on the tab you choose to export, only when you click the button.
  • storage — to cache the configuration file locally (above). No conversation data is stored.
  • alarms — to refresh that configuration file periodically.
  • downloads — to save the exported Markdown file to your Downloads folder when you click "Download .md". Used only for that file, only when you click that button.
  • host access to the supported AI-chat sites — to read a conversation when you click export on that site; and to raw.githubusercontent.com for the config file.

The extension only acts when you open the popup and click export. It does not run in the background on your conversations.

Data sharing

None. We share nothing with anyone because we collect nothing.

Changes

If this policy ever changes, the "Last updated" date above will change.
